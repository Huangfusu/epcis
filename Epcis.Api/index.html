Epcis Repository

This website implements the GS1 EpcGlobal EPCIS specification 1.2.

Endpoints

  • Capture data: /EpcisCapture/
  • Query data (SOAP endpoint): /EpcisQuery/

You can get the WSDL file here: /EpcisQuery?wsdl

Technology Stack

  • Soap services developped usin WCF, C# and .NET framework 4.5.2
  • Database: SQL Server 2008 or higher (must support XML fields)
  • FluentMigrator (Github) for database migrations
  • Quartz.NET (Github) for subscriptions management
  • 827 lines of code (core only - without custom queries)
